# Numeris v3.0 - User Guide

Numeris is an AI-powered financial intelligence platform for market analysis, portfolio tracking, risk review, geopolitical monitoring, and agent-assisted research.

Run locally with:

```bash
scripts/start_all.bat
```

Production uses a Vercel-hosted Vite frontend, a separately deployed FastAPI backend, and Supabase Auth/Postgres for identity and persistence.
